# FortiGate VPN Administration

## Introduction

This project documents my experience managing FortiGate VPN services for remote users in a Windows-based enterprise environment.

The objective was to provide secure remote access to internal company resources while maintaining proper user authentication, access control, and network security.

This project includes VPN user administration, FortiClient configuration, remote access troubleshooting, and user support activities.
